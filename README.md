# TrabalhoFinalProjeto



#Introdução#
Este documento descreve o projeto do Sistema Biblioteca Digital, cujo objetivo é permitir o gerenciamento de acervo, empréstimo/retorno de materiais, cadastro de usuários, e consultas online a partir de uma interface web responsiva. O sistema atende bibliotecas acadêmicas e comunitárias que necessitem de controle digital de exemplares físicos e recursos digitais (ebooks, artigos).
O documento segue o template recebido e descreve atores, casos de uso, modelos de projeto (arquitetura, componentes, classes), diagramas comportamentais e o modelo de dados.
